API Information

Your authentication token is stored in localStorage. You can now use the following API endpoints:

GET /api/v1/tasks - Get all tasks POST /api/v1/tasks - Create a new task PUT /api/v1/tasks/{task_id} - Update a task DELETE /api/v1/tasks/{task_id} - Delete a task POST /api/v1/users/refresh-token - Refresh access token
